# 🎮 Steam Store – Proyecto Full Stack

Proyecto integrador **Full Stack** desarrollado para **NUCBA**.

La aplicación simula una tienda de videojuegos estilo **Steam**, con un **frontend en React** y un **backend en Node.js**, conectados a una **base de datos MySQL**.

---

##  Tecnologías utilizadas

### Frontend
- React
- Vite
- CSS
- Fetch API

### Backend
- Node.js
- Express
- MySQL
- JWT (jsonwebtoken)
- bcrypt
- dotenv

---

##  Funcionalidades

- Listado de videojuegos desde base de datos MySQL
- Carruseles de productos por categoría:
  - Descuentos
  - Lanzamientos
  - Todos los juegos
- Registro de usuarios
- Login de usuarios
- Autenticación mediante JWT
- Endpoints protegidos
- Creación de órdenes de compra
- Conexión completa Frontend ↔ Backend ↔ Base de datos
